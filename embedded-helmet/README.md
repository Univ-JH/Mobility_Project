# embedded-helmet

Arduino Nano 33 BLE(헬멧) 기반 임베디드 영역이다. 압력 센서로 착용 여부를 판정하고, 기울기/가속도 기반 사고/응급 의심 이벤트를 생성한 뒤 BLE로 이동장치(라즈베리파이)에 송신한다.

## 호환성
- BLE/엣지 연동 메시지(telemetry/event로 이어짐)는 백엔드의 v1 MQTT 계약을 기준으로 필드 의미를 맞춘다.
- 안전 이벤트는 `timestamp/seq/rideId`를 포함하도록 설계한다.

## 확장성
- 센서 추가/교체 시에는 이벤트 생성 인터페이스만 유지한다.

## 가시성/명확성
- 이벤트 생성 시점의 원천 값(가능하면)과 최종 이벤트 라벨을 분리해 추적한다.
