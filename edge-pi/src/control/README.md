# edge-pi/src/control

서보 모터를 이용한 제동/속도 제한 제어 로직을 담당한다.

## 호환성
- 서버에서 내려온 `ttlMs` 내에만 제어 명령을 적용한다.
- 단계형 제어(`warn -> limit -> brake -> emergency`) 정책을 따른다.

## 확장성
- 제동 캘리브레이션 테이블/파라미터는 코드 상수로 박지 않고 설정/테이블에서 읽도록 확장한다.

## 가시성/명확성
- 제어 명령 생성 시 반드시 `commandId`/`reason`/`severity` 관측 필드를 유지한다.

## ===============================================================================================

##  파일 구성
1. `config.json`: 핀 번호 및 제동 강도 설정
2. `brake_system.py`: 서보 모터 부드러운 제동 구현
3. `sensor_manager.py`: 거리 및 헬멧 데이터 수집
4. `vision_ai.py`: YOLOv8 도로/인도 판별
5. `main_app.py`: 통합 상황 판단 및 제어

##  제어 우선순위
1. **CRITICAL**: 헬멧 미착용/사고 (EMERGENCY 제동)
2. **HIGH**: 장애물 근접 (BRAKE 제동)
3. **MEDIUM**: 인도 주행 (LIMIT 제동)
4. **LOW**: 정상 주행 (RELEASE 해제)
