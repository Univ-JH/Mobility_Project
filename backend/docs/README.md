# backend/docs

백엔드 세부 문서(구현 전/후 모두 포함) 영역이다.

## 포함 권장
- MQTT v1 payload 스키마/필드 정의(타 언어/펌웨어용 템플릿 포함)
- 상태 전이 매트릭스/전이 근거 이벤트 규칙
- 정책 DSL의 확장 방법(버전/호환성 규칙 포함)

## 호환성/확장성
- 문서는 `.cursor/rules`에 반영될 때까지 코드에 강제되지 않는다.
- 문서 버전은 가능하면 날짜/버전을 함께 남긴다.

## 가시성/명확성
- “문서 계약”은 변경 이력(Changelog) 또는 날짜 라벨을 통해 추적 가능해야 한다.
